28. 实现 strStr()

2020-03-22 / 3 min read

一、题目

实现 strStr() 函数。
给定一个 haystack 字符串和一个 needle 字符串，在 haystack 字符串中找出 needle 字符串出现的第一个位置 (从0开始)。如果不存在，则返回 -1。

示例 1:

输入: haystack = "hello", needle = "ll"
输出: 2

示例 2:

输入: haystack = "aaaaa", needle = "bba"
输出: -1

说明: 当 needle 是空字符串时，我们应当返回什么值呢？这是一个在面试中很好的问题。
对于本题而言，当 needle 是空字符串时我们应当返回 0 。这与C语言的 strstr() 以及 Java的 indexOf() 定义相符。

二、解决方法

废话不多说，先上代码。
最初的实现（通过）：

int strStr(char* haystack, char* needle) {
    char* phk = haystack;
    char* pne = needle;
    int index = 0;  // 当前指针phk指向的元素的下标，与指针phk绑定（所谓绑定就是二者同增同减）
    int cnt = 0;    // 纪录回溯时需要后退的间隔数，与指针pne绑定
    
    if (*pne == '\0')
        return 0;
    while (*phk != '\0')
    {
        if (*phk != *pne)
        {
            phk++;
            index++;
        }            
        else // the cur two pointer's value is equal
        {
            phk++;
            index++;
            pne++;
            cnt++;
            if (*pne == '\0' && *phk == '\0')
                return 0;
            while (*pne != '\0' && *phk != '\0')
            {
                if (*pne != *phk)
                {
                    phk++;
                    index++;                    
                    phk -= cnt;                    
                    index -= cnt;
                    pne -= cnt;
                    cnt -= cnt;
                    break;
                }                    
                else
                {
                    pne++;
                    cnt++;
                    phk++;
                    index++;
                }
                if (strlen(pne) > strlen(phk))
                    return -1;
            } 
            if (*pne == '\0')
            {
                if (*phk == '\0')
                    return index - cnt;
                else
                    break;
            }               
        }
    }
    if (index == strlen(haystack))
        return -1;
    return index - cnt;
}

思路：设置两个指针分别指向两个字符串头部，然后从字符串 haystack 头部往后遍历，去寻找其中与字符串 needle 相同的子串。这个遍历的过程中最要紧的是要注意回溯，例如对于字符串
haystack[] = "aaaba"，needle[] = "aab"，当两个指针都指到字符串中第三个字符时，此时字符a不等于字符b，这时我们不能把指向 haystack 的指针往后移，而是应该将指针回溯到第二个字符，再重新开始比较。由于 cnt 纪录了当前字符串 needle 已经匹配了的字符的个数，因此需将字符串 haystack 的指针 phk 后移 cnt 个间隔，这样便完成了回溯。
实现回溯的代码如下：

if (*pne != *phk)
{
    phk++;
    index++;                    
    phk -= cnt;                    
    index -= cnt;
    pne -= cnt;
    cnt -= cnt;
    break;
}
