26. 删除排序数组中的重复项

2020-03-04 / 2 min read

😄From here, I start my journal on LeetCode.
👉Let's practice programming together!
😇Hoping you could give some sugesstions.

一、题目

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

示例 1：

给定数组nums = [1,1,2], 
函数应该返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2。
你不需要考虑数组中超出新长度后面的元素。

示例2：

 给定 nums = [0,0,1,1,1,2,2,3,3,4],
函数应该返回新的长度 5, 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4。
你不需要考虑数组中超出新长度后面的元素。

二、解决方案

代码：

int removeDuplicates(int* nums, int numsSize)
{
    // 数组只有一个元素或者为空，直接返回原长度
    if (numsSize <= 1)
    {
        return numsSize;
    }
    int length = 0;     // 初始化最终数组长度
    for (int i = 1; i < numsSize; i++)  // 从第二个元素所在位置开始遍历数组
    {
        if (nums[i] != nums[length])    // 如果当前元素与length所在位置元素不相等
        {
            nums[++length] = nums[i];   // 当前元素的值赋给length所在位置的后一个元素
        }
    }
    return length + 1;
}
